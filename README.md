# VRChat Image Converter

[VRCImageHelper by m-hayabusa](https://github.com/m-hayabusa/VRCImageHelper)に強くインスパイアされたツールです。  

> [!WARNING] ログ出力を"完全"にしてください
> VRChatのログ出力がオフ、もしくはエラーのみになっていると動作しません。

## サポートされているプラットフォーム

|プラットフォーム|対応状況|
|---|---|
|Windows|❓️(情報提供お待ちしております)|
|Linux|✅️|
